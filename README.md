# 🚀 Sistema de Gestión de Incidencias

Aplicación web desarrollada para registrar y gestionar incidencias técnicas en entornos de telecomunicaciones.

## 🌐 Demo Online

👉 https://gestion-incidencias-fdo2.onrender.com

---

## 📌 Funcionalidades

- ✅ Crear incidencias
- ✅ Listar incidencias
- ✅ Actualizar estados
- ✅ Eliminar incidencias
- ✅ Base de datos SQLite
- ✅ Backend con Node.js y Express

---

## 🛠️ Tecnologías utilizadas

- HTML
- CSS
- JavaScript
- Node.js
- Express
- SQLite
- Git / GitHub


## ⚙️ Instalación local

```bash
git clone https://github.com/Maffares89/gestion-incidencias.git
cd gestion-incidencias
npm install
node server.js
```
